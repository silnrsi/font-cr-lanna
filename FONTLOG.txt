FONTLOG for the CR fonts

This file provides detailed information on the CR Font Software. This information should be distributed along with the CR fonts and any derivative works.

Basic Font Information

CR is a Unicode typeface family that supports Tai Tham in the Lanna style. The family contains the following fonts:

Cr-Udom - Latin from Lato-Regular
